# Technical Stack

## Source Fingerprint

- Hosting: Vercel
- Framework/runtime: Next.js App Router and React
- Rendering: static HTML plus client-side React chunks
- 3D runtime: Three.js / React Three Fiber chunks
- Analytics references: Google Analytics and Vercel Analytics

## Local Stack

- Node.js 24 or newer
- No runtime npm dependencies
- Static files served directly from the repository root

## Important Paths

- `index.html`: captured home page
- `_next/static/`: captured Next.js CSS and JavaScript chunks
- `portfolio/`: replaceable project images
- `project*.png`: replaceable project cover textures
- `font/`: Brockmann font files
- `fav/`: icons and web manifest
